# 数据结构与算法

往往有人把数据结构与算法混为一谈，以为学习数据结构就是在学习算法，或者以为学习算法
就是在学习数据结构，这样的理解是不对的。

数据结构与算法属于两种不同的学科，数据结构主要讲的是数据存储的问题，而算法主要讲的
是数据处理的问题，这种学科的内容是不一样的。不过既然都涉及都数据层面，
且有些数据结构是随着某种算法的诞生而被发明出来的，所以前辈计算机科学家们把它们归为一门课程。
